# Aura - Vehicle Marketplace Platform

A comprehensive vehicle marketplace platform built with React, TypeScript, and Supabase.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📁 Project Structure

```
Abhi-auraVite-ts/
├── src/                    # Source code
│   ├── components/         # React components
│   ├── pages/             # Page components
│   ├── hooks/             # Custom React hooks
│   ├── api/               # API integration
│   ├── lib/               # Utility libraries
│   └── utils/             # Helper functions
├── database-migrations/   # SQL files and documentation
├── public/                # Static assets
└── config files          # Build and configuration
```

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **State Management**: React Hooks
- **Forms**: React Hook Form + Zod
- **Icons**: Lucide React
- **Charts**: Recharts
- **Maps**: Leaflet + React Leaflet

## 🔧 Key Features

- **Authentication**: Email/password with verification
- **Vehicle Management**: Add, edit, view vehicles
- **Marketplace**: Browse and search vehicles
- **Dealer Dashboard**: Business management tools
- **Admin Panel**: System administration
- **Real-time Updates**: Live data synchronization
- **Document Management**: File uploads and storage
- **Analytics**: Performance tracking and insights

## 📊 Database

All database migrations and documentation are in the `database-migrations/` folder. See the README there for detailed information.
